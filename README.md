# Mitski Songs App

> This project is currently under construction.

A cozy pixel art desktop music player built with Electron, featuring a selection of Mitski songs.

## Preview

<img width="466" height="587" alt="image" src="https://github.com/user-attachments/assets/169d65a5-8719-479c-8461-95505241037e" />


## Features

- Desktop application built with Electron
- Custom pixel art interface
- Play and pause controls
- Previous and next song controls
- Song cover images
- Music progress bar
- Custom window controls
- Custom pixel art cursor

## Technologies

- HTML
- CSS
- JavaScript
- Electron
- Node.js
